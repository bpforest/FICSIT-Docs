= 개발

자신만의 모드를 만들고 싶은 것이라면, 이 섹션이 적합할 것 같군요!

여기에서 모드를 만들고 개발 환경을 설정하고 실행하는 방법에 대한 정보를 확인할 수 있습니다.
시작하려면 왼쪽의 하위 카테고리를 확인하십시오.

그저 새티스팩토리와 모드를 즐기고 싶은 것이라면, 대신 xref:index.adoc#_사용자용[여기]에서 모드를 설치하는 방법을 확인해 보십시오.
